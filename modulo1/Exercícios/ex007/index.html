Semântica ou Não Semântica

Negrito / Destaque

A primeira forma de adicionar negrito em uma frase é adiocionando a tag <B>(não semântica)

A segunda que é a forma mais utilizadano HTML5 é a tag <STRONG> (semântica)

Itálico / Ênfase

A forma NÃO SEMÂNTICA se utliza a tag <i>

Agora a forma SEMÂNTICA utiliza-se a tag <em>

Marcando texto

Para marcarmos um texto usamos a tag <MARK>

Texto Grande e Pequeno

Para colocarmos o texto em formato GRANDE podemos utilizar o CSS ou a tag <big>

Para colocarmos o texto em formato PEQUENO podemos utilizar o CSS ou a tag <small>

Textos deletados

Textos deletados não tem uma importância muito grande, para adiconar usamos a tag <del>

Textos inseridos

Os textos inseridos são bastantes utlizados nos sites, para adicionar usamos a tag <ins>

Textos sobrescritos

Texto sobrescrito tem a função de adiocionar letras ou numeros em CIMA da palavra, para adicionar utilizamos a tag <sup>
como por exemplo: x55+L2>

textos subescritos

Textos subescritos tem a função de adicionar letras ou números em BAIXO da palavra, para adicionar utilizamos a tag <sub>
como por exemplo: H2O; CH2
